# postman-environment-generator
A library that generates environments for postman using a simple configuration


How to include in your project?

For Maven:

```
    <dependency>
        <groupId>com.darkestapp</groupId>
        <artifactId>postman-environment-generator</artifactId>
        <version>0.1.0</version>
        <type>pom</type>
    </dependency>
```


For gradle:


```
compile 'com.darkestapp:postman-environment-generator:0.1.0'
```
